# Sticks

Some building material for web apps.

## Usage

- Compile HTML files into `_site`:

```
$ make
```

- Run a stand-alone server:

```
$ make run
```

### Requirements

- [Jekyll](http://jekyllrb.com/)
- [LessCSS](http://lesscss.org/)
- [YUI Compressor](http://developer.yahoo.com/yui/compressor/)
